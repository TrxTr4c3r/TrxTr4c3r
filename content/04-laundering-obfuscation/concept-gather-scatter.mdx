# Gather-Scatter

<span style={{ fontFamily: 'var(--font-space-mono), monospace', fontSize: 13, opacity: 0.7 }}>`concept`</span>

## Core idea
Combines fan-in and fan-out in sequence: many small amounts are gathered into one wallet, then immediately scattered back out to a different set of wallets, severing the link between original depositors and eventual recipients.

## When to use
A wallet with a short-lived spike in both inbound and outbound transaction count and near-zero holding time.

## Example
Dozens of mule deposits converge on one wallet, which within minutes fans them out to a fresh, unrelated set of addresses.
