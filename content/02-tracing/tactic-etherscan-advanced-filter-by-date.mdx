# Etherscan Advanced Filter for Point-in-Time Review

<span style={{ fontFamily: 'var(--font-space-mono), monospace', fontSize: 13, opacity: 0.7 }}>`tactic`</span>

## Core idea
Etherscan's Advanced Filter lets you restrict an address's transactions to a date range. Filtering up to the day of an event reconstructs exactly what was visible on that address before the event, which is powerful for showing what due diligence would have revealed.

## When to use
When you need to demonstrate what someone could or should have seen about an address at a specific past moment, such as before hiring a developer or before an exploit.

## Example
Filtering a developer's address to the period before an exploit showed outbound transfers to an OFAC-sanctioned address, evidence the protocol could have found with basic due diligence before employing him.
